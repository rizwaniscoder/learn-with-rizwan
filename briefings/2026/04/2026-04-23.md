  *Who can act on it:* Claude integration specialists, RAG/eval shops, prompt-engineering + MCP/A2A builders.
  *Why now:* the 3.5 GW TPU expansion through Broadcom (from 2027) locks in years of price-competitive Claude capacity — enterprises are betting long.
  *Angle:* package a "Claude-to-production" offer — MCP server build + eval harness + agent observability — and pitch to any shop already on AWS Bedrock.
  [source](https://www.anthropic.com/news/google-broadcom-partnership-compute)

- **Voice agents eating GCC contact-center outsourcing — 60-80% cost savings quoted.**
  *What happened:* the Middle East contact center outsourcing market (projected $3.27B by 2031) is rapidly being disrupted by voice AI agents delivering 60-80% cost savings while maintaining 24/7 availability.

  *Why now:* GCC BPO growth is 10%+ annually, and incumbents are the most exposed; local-language voice AI (Arabic dialect fine-tunes) is still underbuilt.
  *Angle:* pitch a "voice AI pilot in 30 days" to mid-sized GCC telcos and banks — use Retell AI or Vapi as the stack, charge per successful conversation.
  [source](https://www.strategyand.pwc.com/m1/en/industries/technology/high-growth-opportunity.html)

- **Factory's $1.5B raise tells enterprises are buying coding *agents*, not assistants.**
  *What happened:* Factory raised $150M Series C led by Khosla at $1.5B valuation on the strength of "Droids" — agents that handle testing, review, docs, and deployment, not just code completion. Customers include Nvidia, Adobe, EY, Palo Alto Networks.
  *Who can act on it:* custom-agent builders for internal dev workflows, CI/CD + agent integration consultancies.
  *Why now:* Factory is proof a coding-agent rollout can be sold as a services engagement at enterprise scale.
  *Angle:* offer "agent-ops for engineering" retainers — monthly fee for running and tuning coding agents inside one customer's SDLC.
  [source](https://techcrunch.com/2026/04/16/factory-hits-1-5b-valuation-to-build-ai-coding-for-enterprises/)

Zuckerberg committed up to $135B in 2026 capex. The goal: train agents that can operate dropdowns and keyboard shortcuts — the last 20% of knowledge work automation.

## 🛠️ New tools gaining traction

- **Hermes Agent** — self-improving agent using DSPy + Genetic Evolution Prompt Architecture (GEPA) to extract and refine skills from conversations.
  *Traction:* +32,572 GitHub stars in one week; #1 on GitHub trending (week of Apr 13).
  *Why it matters:* first agent framework with an actual self-improvement loop that doesn't collapse — opens the door to long-running agents that improve without retraining.
  [source](https://www.shareuhack.com/en/posts/github-trending-weekly-2026-04-13)

- **MemPalace** — open-source AI memory system, 96.6% on LongMemEval.
  *Traction:* 20K+ GitHub stars in 48 hours after April 5 launch; fastest-growing new AI repo this month.
  *Why it matters:* highest-scoring free memory layer — drops the cost of persistent-memory agent deployments.
  [source](https://www.shareuhack.com/en/posts/github-trending-weekly-2026-04-13)

- **MiniMax M2.7** (open weights) — 230B MoE, 10B active, 256-expert self-evolving agent model.
  *Traction:* 56.22% on SWE-Pro (matching GPT-5.3-Codex); 1,495 Elo on GDPval-AA (top open-source); integrated into NVIDIA platforms and Ollama.
  *Why it matters:* an agent that demonstrably self-improved its own scaffold (30% lift on internal evals) — the first credible step past static fine-tuning.
  [source](https://www.marktechpost.com/2026/04/12/minimax-just-open-sourced-minimax-m2-7-a-self-evolving-agent-model-that-scores-56-22-on-swe-pro-and-57-0-on-terminal-bench-2/)
