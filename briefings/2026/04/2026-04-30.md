## 🛠️ New tools gaining traction

- **OpenClaw** — local-first open-source AI agent framework that runs entirely on user devices (macOS/iOS/Android), with a live Canvas and shell/file/web automation.
  *Traction:* 347K GitHub stars by late April (302K by April 3) — the fastest-growing repo in GitHub history; peak velocity 12K stars/day; broke GitHub's "AI" trending algorithm.
  *Why it matters:* local-first agent execution undercuts the cloud-agent SaaS pricing model — privacy-conscious enterprises now have a credible self-hostable option.
  [source](https://thesoogroup.com/blog/openclaw-github-phenomenon-autonomous-agent-framework)

- **Cursor 3** (Apr 2) — IDE rewrite around an Agents Window that runs many agents in parallel across local, worktrees, cloud, and remote SSH; adds Design Mode for browser-targeted UI feedback and multi-repo workspaces.
  *Traction:* every major AI tool tracker listed it as the biggest dev-tool release of April; integrated Claude Opus 4.7, GPT-5.5, and DeepSeek V4 within a week.
  *Why it matters:* the IDE is no longer one human + one assistant — it's one human supervising N agents. Pricing and team-seat economics change.
  [source](https://cursor.com/blog/cursor-3)

- **Workspace Agents in ChatGPT** (Apr 22) — OpenAI's enterprise successor to custom GPTs; Codex-powered, native integrations with Slack, Google Workspace, Microsoft 365, Salesforce, Notion, Atlassian Rovo.
  *Traction:* free until May 6, then credit-priced; rolling out to Business/Enterprise/Edu plans; positioned as a direct response to Claude's Projects + MCP.
  *Why it matters:* the custom-GPT era is dead — enterprises will rebuild agents on this stack, and the ones who built on the old GPT Store now need migration help.
  [source](https://openai.com/index/introducing-workspace-agents-in-chatgpt/)
