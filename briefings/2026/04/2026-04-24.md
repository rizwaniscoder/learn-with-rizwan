## 🎯 Opportunity signals
*For founders and service businesses.*

- **GPT-5.5's agent-first pricing forces every enterprise on GPT-5.4 to re-cost their stack this quarter.**
  *What happened:* OpenAI released GPT-5.5 on April 23 at 2× the input/output price of 5.4 ($5/$30 per 1M tokens), but claims ~40% fewer output tokens per task, netting to roughly a 20% cost increase for the same workload. Pro pricing is unchanged at $30/$180.
  *Who can act on it:* LLM-cost-optimization shops, prompt-efficiency consultants, model-routing specialists (LiteLLM, Portkey, OpenRouter integrators).
  *Why now:* CFOs will ask why the AI bill went up 20%+ overnight. Selling a "GPT-5.5 migration + eval + cost benchmark" as a 2-week engagement is an easy door-opener.
  *Angle:* cold outreach to any Series B+ AI-forward company: "free 48-hour GPT-5.5 vs 5.4 cost-performance audit on your top three prompts."
  [source](https://openai.com/index/introducing-gpt-5-5/)

  *Why now:* CVC's forward-deployed Googlers can't cover 100+ portfolio companies — they'll sub-contract. This channel opens through May.
  *Angle:* list CVC's current portfolio on CVC.com, pick 5 targets in your vertical, offer a Gemini Enterprise PoC for their top process pain; reference the Google partnership in subject line.

  *Who can act on it:* BI consultancies, dbt/LookML shops, MCP-server builders, data-contract specialists.
  *Why now:* every "my ChatGPT agent got the wrong revenue number" incident is now an executive-level problem. Budget is opening.
  *Angle:* package a "Semantic-Layer-for-AI-Agents audit" for mid-market: 1-week diagnostic, then an Omni/dbt Semantic/Cube.dev implementation. Refer out, take the services retainer.

  *Why now:* Pichai's disclosure that 75% of Google's new code is AI-generated forces every CISO to ask the same question about their own stack.

## 🛠️ New tools gaining traction

- **ml-intern (Hugging Face)** — open-source agent that automates LLM post-training end-to-end: literature review, dataset discovery, training script execution, iterative eval. Built on smolagents.
  *Traction:* in the launch demo, improved Qwen3-1.7B on GPQA from ~10% to 32% in under 10 hours, beating Claude Code's 22.99% on the same task; launched April 21.
  *Why it matters:* first open agent credibly automating the full post-training loop — services shops can stop charging by the hour for fine-tuning setup and charge by the outcome instead.
  [source](https://www.marktechpost.com/2026/04/21/hugging-face-releases-ml-intern-an-open-source-ai-agent-that-automates-the-llm-post-training-workflow/)

- **andrej-karpathy-skills (CLAUDE.md)** — a single CLAUDE.md file encoding Karpathy's observed LLM-coding pitfalls as enforceable rules.
  *Traction:* on the order of 30k+ GitHub stars, the core of a wider "Skills" framework ecosystem; +44K weekly stars on the broader skills cluster per the GitHub Trending Weekly report for Apr 22.
  *Why it matters:* teams are standardizing on shared agent etiquette. If you deliver Claude Code or Cursor rollouts, ship this as your baseline CLAUDE.md on day one.
  [source](https://github.com/forrestchang/andrej-karpathy-skills)

- **OpenClaw** — open-source personal computer-use agent (web, shell, form-fill, self-authored skills).
  *Traction:* ~210,000+ GitHub stars in under four months — among the fastest-growing OSS repos ever; now spun into an independent foundation after creator Steinberger joined OpenAI in Feb.
  *Why it matters:* the open-source benchmark for computer-use agents — useful for offline/on-prem clients who can't ship screens to a frontier lab.
  [source](https://fortune.com/2026/02/19/openclaw-who-is-peter-steinberger-openai-sam-altman-anthropic-moltbook/)
