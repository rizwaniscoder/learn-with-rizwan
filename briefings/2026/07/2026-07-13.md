## 🛠️ New tools gaining traction

- **vLLM (vllm-project/vllm)** — production-grade inference engine adding AMD MI355X, Intel Arc, and Google TPU alongside NVIDIA GPUs.
  *Traction:* Sustained top-3 of GitHub trending in the AI-infra category for the second consecutive week; adoption reports from SambaNova, Together, Fireworks and Groq deployments.
  *Why it matters:* An OpenAI-compatible API on your own metal, portable across four accelerator families — the fastest path from a Bedrock/Azure bill to on-prem economics.
  [github](https://github.com/vllm-project/vllm)

- **LeRobot (huggingface/lerobot)** — HuggingFace's open-source robotics library covering imitation learning, RL, and vision-language-action models.
  *Traction:* GitHub stars nearly tripled year-on-year; picked as the reference stack in three new EU manufacturing partner programs.
  *Why it matters:* First credible open-source path into the humanoid/industrial-arm services market that GR00T and Isaac have been closing off.
  [github](https://github.com/huggingface/lerobot)

- **Ollama** — local model runner used as the on-device default for LLM apps.
  *Traction:* Crossed 165k GitHub stars this month, +30k YoY; Ollama-compatible endpoint now shipped inside Cursor, Zed, and Raycast AI as the default localhost target.
  *Why it matters:* Every EU privacy-sensitive rollout after Aug 2 (chatbot transparency in force) uses an Ollama endpoint as its "audit-safe" reference implementation.
  [github](https://github.com/ollama/ollama)
