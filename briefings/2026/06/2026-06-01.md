## 🛠️ New tools gaining traction

- **Sakana Fugu** — commercial multi-agent orchestration service from Sakana AI; a 7B RL-trained "Conductor" routes tasks across GPT-5, Claude Sonnet 4 and Gemini 2.5 Pro behind a single OpenAI-compatible API. Two tiers: Fugu Mini (low latency) and Fugu Ultra (max performance).
  *Traction:* beta opened April 25; covered by VentureBeat, MarkTechPost and Sakana's own blog; the underlying ICLR 2026 Trinity/Conductor papers are the most-cited multi-agent work this cycle. [Sakana](https://sakana.ai/fugu-beta/) · [VentureBeat](https://venturebeat.com/orchestration/how-sakana-trained-a-7b-model-to-orchestrate-gpt-5-claude-sonnet-4-and-gemini-2-5-pro)
  *Why it matters:* if a 7B router beats GPT-5 alone on the same task, "model-routing as a service" becomes a category — and every agent builder will rip out hard-coded LLM calls within the quarter.

- **Vibe (Mistral)** — relaunched Le Chat with Work Mode and remote async coding agents, all running on Mistral Medium 3.5 (128B dense, 256k context, $1.50/$7.50 per million in/out tokens). Work Mode coordinates email, calendar, Jira, GitHub and Slack with explicit approvals; remote Vibe agents handle long-running coding tasks.
  *Traction:* shipped April 30; Mistral Le Chat downloads spiked 4× in May, and AI Twitter is using "Vibe coding" as a generic term for the workflow. [Mistral](https://mistral.ai/news/vibe-remote-agents-mistral-medium-3-5/)
  *Why it matters:* an EU-sovereign agent stack with usable async coding is finally credible — Schwarz-backed Cohere/Aleph-Alpha and Vibe together give regulated enterprises two non-US options.

- **Grok Build** — xAI's coding CLI for SuperGrok and X Premium Plus subscribers; runs from the terminal, agentic edits and shell exec, integrates with Grok Imagine for screenshot-driven debugging.
  *Traction:* opened in beta mid-May; AI engineering Twitter and r/LocalLLaMA picked it up; xAI ships it ahead of the SpaceX-IPO roadshow window that opens June 4. [xAI release notes](https://releasebot.io/updates/xai)
  *Why it matters:* xAI is the first lab to ship a Cursor-style CLI without a separate IDE — pricing is bundled into SuperGrok so the per-token cost looks like zero, which will dent Cursor and Claude Code's mid-market on procurement.
